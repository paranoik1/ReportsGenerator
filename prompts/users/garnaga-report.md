Необходимо доработать отчёт по лабораторной работе на основе файлов:

* «1 практическая.docx» — содержит недоделанный отчёт с неполной структурой; его необходимо завершить на основе практического задания. Опирайся на его форматирование
* «Lab_1.pdf» — содержит задание для лабораторной работы (включает пример выполненной работы; если в «1 практическая.docx» отсутствуют какие-либо пункты, их необходимо добавить).
* «Практическая 2.docx» — содержит индивидуальную тему.

Задача:

* Заполнить все таблицы в отчёте по основной теме из «Lab_1.pdf».
* В пункте 4 «Построение даталогической модели БД» вставить изображение (если путь не указан — использовать заглушку).
* Из практической работы №2 взять только тему.
* Для данной темы выполнить полный аналогичный анализ, как в лабораторной работе №1:

  * анализ предметной области,
  * список сущностей,
  * списки атрибутов таблиц,
  * список связей,
  * инфологическая модель,
  * даталогическая модель.

Результаты анализа по индивидуальной теме необходимо разместить:

* сразу после раздела «Даталогическая модель БД» основной темы,
* перед итоговым выводом по всей работе.

Требования к оформлению заголовков:

* Название лабораторной работы, тема и цель — по центру.
* Цель работы размещается с новой строки после заголовка (не по центру).
* Вывод оформляется аналогично цели.
* Разделы вида «1. Анализ предметной области» не центрируются.

По центру размещаются:

* Список сущностей
* Список атрибутов таблицы «Клиенты»
* Списки атрибутов остальных таблиц
* Аналогичные структурные элементы
