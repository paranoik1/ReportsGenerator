# Переход на Streamlit: Руководство по миграции

## 📋 Обзор изменений

Фронтенд проекта переписан с **Flask + HTML/CSS/JS** на **Streamlit**. Это упрощает разработку новых AI-функций и улучшает пользовательский опыт.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd /workspace
poetry install
```

### 2. Запуск приложения

**Вариант A: Два терминала (рекомендуется для разработки)**

Терминал 1 - Backend (Flask API):
```bash
./run_backend.sh
```

Терминал 2 - Frontend (Streamlit):
```bash
./run_frontend.sh
```

**Вариант B: Один терминал (фоновый режим)**

```bash
# Запуск backend в фоне
./run_backend.sh &

# Запуск frontend
./run_frontend.sh
```

### 3. Доступ к приложению

- **Streamlit UI**: http://localhost:8501
- **Flask API**: http://127.0.0.1:5000

## 🔧 Конфигурация

### Переменные окружения

Создайте файл `.env` в корне проекта:

```bash
# API Base URL (по умолчанию)
API_BASE_URL=http://127.0.0.1:5000

# Настройки Streamlit (опционально)
STREAMLIT_SERVER_PORT=8501
STREAMLIT_SERVER_ADDRESS=localhost
```

1. **Буфер обмена**: Прямая вставка из Ctrl+V требует дополнительной работы (browser permissions)
2. **Производительность**: При большом количестве изображений (>20) возможна задержка рендеринга

## 🆘 Решение проблем

### Ошибка "Server unavailable"

1. Проверьте, запущен ли Flask backend
2. Убедитесь, что порты не заняты
3. Проверьте переменную окружения `API_BASE_URL`

### Ошибка импорта Streamlit

```bash
poetry add streamlit pillow requests
```

### Зависание polling

- Обновите страницу (F5)
- Нажмите "🔄 Сбросить форму" в sidebar

## 📊 Сравнение подходов

| Характеристика | Flask + JS | Streamlit |
|----------------|------------|-----------|
| Строк кода | ~660 (HTML+JS+CSS) | ~680 (Python) |
| Язык | HTML/CSS/JS + Python | Только Python |
| Время разработки | Дольше | Быстрее |
| Кастомизация дизайна | Полная | Ограниченная |
| Real-time обновления | WebSocket/SSE | Polling/Rerun |
| Мобильная поддержка | Полная | Ограниченная |
| Интеграция с AI | Средняя | Отличная |
