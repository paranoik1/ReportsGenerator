# Roadmap

## Проблемы
- LLm плохо справляется с объемными практическими работами:
    - вставляет изображения в непопад
    - Решение: сокращение контекста и разделение информации на блоки данных, к которым ИИ агент сможет обращаться, когда нужно 
- Изображения вставляются неверно - Обтекания нет и Привязка к символу должна быть (https://docs-python.ru/packages/modul-python-docx-python/rabota-risunkami/)


## Будущие функции проекта:
- Анализ изображений документов пользователя
- Кэширование результатов LLM (например, если на вход doc_analyz поступает документ, который был ранее - возвращает ответ, который был сгенерирован ранее)
