# Roadmap

## Formatter CoT (Chain-of-THought)
- Прочитать блоки данных, связанные с практической работой
- Выяснить структуру отчета и как форматировать финальный отчет
- Начать писать отчет по этапно (не весь сразу, а по секциям)
    - Читает нужные блоки данных (вызывая read_block tool с параметром id блока - paralel_run_tools) для получения необходимой информации для написания конкретной части/секции/чанка отчета 
    - Пишет секцию, отчет
- Как только отчет написан, необходимо вызывать finish tool

    - Сохранение state в json после ошибок на верхнем уровне
